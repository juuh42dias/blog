It’s a shortcut for use: (instead of)

```
def self.method
end

def self.other_method
end
```

You don’t need to write `self` all the time, so, you can use:

```
class << self

def method
 end

def other_method
 end
end
```

So, it’s DRY, removing repetitions on your code.
